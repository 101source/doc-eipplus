工作紀錄¶

規格說明¶

  • 提供組織內部員工將每日工作紀錄。
  • 支援權限管控，對應之主管或人員可以看到部屬或相關人員之工作紀錄。
  • 工作紀錄可以上傳檔案。
  • 可以紀錄工時並作統整。
  • 可以篩選工作紀錄，並將資料匯出。
  • 提供高階管理人以月的方式快速檢視部門或人員工作回報紀錄。

系統設定¶

設定分類¶

  1. 管理者可在系統管理區塊，點選 設定分類 進行設定。
  2. 點選 新增，產生新的分類。
  3. 在新增分類頁面中，把資料填入相對應欄位，設定完成後點選 儲存。

系統進階搜尋¶

  1. 管理者可在系統管理區塊，點選 系統進階搜尋 進行設定。
  2. 再搜尋頁面中，把資料填入相對應欄位，設定完成後點選 搜尋。

圖表 1.5 系統進階搜尋系統畫面

工作狀態設定¶

  1. 管理者可在系統管理區塊，點選 工作狀態設定 進行設定。
  2. 在設定頁面中，把資料填入相對應欄位，設定完成後點選 儲存。 ( 或 圖示可新增或移除資料夾。)
